# MSIS-522-Assignment-1-

# COVID-19 Patient Mortality Risk Analysis

## Project Overview
This project uses a dataset of over 1 million anonymized patient records to predict COVID-19 mortality risk based on 17 features, including age and pre-existing comorbidities. 

## Deliverables
- **Notebook/Scripts:** `msis_522_hw1.py` contains the full analysis and model training code.
- **Streamlit App:** `app.py` contains the code for the web interface.
- **Models:** Saved `.pkl` and `.keras` files for Logistic Regression, Decision Tree, Random Forest, LightGBM, and MLP.

## How to Run Locally
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
